Cloud Vision APIで画像解析をする
=========================================================

Vision APIで画像の解析をやってみます。Botに送信した画像をVision APIでどんな画像かを返信します。

`Vision API <https://cloud.google.com/vision/?hl=ja>`_

Vision APIは対象の画像を解析して、画像のいろいろな情報を取得できます。

- 物体検知
- 有害コンテンツ検知
- ロゴ検知
- ランドマーク検知
- OCR
- 顔検知
